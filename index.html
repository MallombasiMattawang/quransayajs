QuranSaya.JS


QuranSaya.Js merupakan sebuah plugin jQuery, yang dapat menampilkan terjemahan ayat-ayat suci Al-Quran pada sebuah website. Disini akan dijelaskan bagaimana cara menggunakan QuranSaya.JS

Download View On Github



Contoh Penggunaan :

Kode seperti Ini:

$('#selector').quransaya({surat: 16, ayat: 23});
//note surat bisa diisi dengan no 1 sampai 114



Contoh Penggunaan Memakai Warna:

//pertama anda harus memakai file quransaya.css terlebi dahulu
<link rel="stylesheet" href="quransaya.css">
$('#selector').quransaya({surat: 16, ayat: 23, warna: 'green'});
